# Contributing

Thanks for contributing to the Karki Aayog Report project.

## What to contribute

- factual corrections
- improved sourcing and citations
- accessibility fixes
- SEO improvements
- visual polish
- performance improvements
- Nepali and English copy improvements

## Ground rules

- preserve factual accuracy
- do not remove source links without replacing them with better ones
- clearly label interpretation versus reported fact
- keep the tone public-interest focused and non-partisan
- avoid adding unverified claims

## Pull requests

Please include:

- a short summary of what changed
- why the change improves the site
- any source links used for factual edits
- screenshots for visible UI changes when relevant

## Content and media

Before adding new media assets, make sure the repository can legally include them and that attribution requirements are satisfied.
